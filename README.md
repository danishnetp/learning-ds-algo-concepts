# learning-ds-algo-concepts

This project is now configured as a Gradle Java application.

## Project structure

- `build.gradle`: Gradle build configuration
- `settings.gradle`: Gradle project name
- `src/main/java`: Application source code
- `src/test/java`: JUnit tests

## Quick start

If Gradle is installed globally:

```powershell
gradle clean test
gradle run
```

If Gradle is not installed, generate wrapper once Gradle is available:

```powershell
gradle wrapper
.\gradlew.bat test
.\gradlew.bat run
```
